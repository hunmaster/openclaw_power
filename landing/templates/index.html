왜 CommentBoost인가?

로컬 실행 기반으로 계정 안전성을 최우선으로 합니다

🚀

Notion 연동 자동화

Notion DB에 영상 링크와 댓글을 입력하면 자동으로 댓글이 작성됩니다. 결과 URL도 Notion에 자동 기록.

🔒

안티디텍트 브라우저

계정마다 고유한 브라우저 지문(핑거프린트)을 생성. IP 변경(ADB 비행기모드) + 프록시 로테이션 지원.

👍

좋아요 자동 구매

댓글 작성 후 SMM API로 좋아요를 자동 주문. 3개 티어(베이직/스탠다드/프리미엄) 선택 가능.

💻

로컬 실행, 내 PC에서

클라우드가 아닌 당신의 PC에서 실행됩니다. 브라우저 세션, 쿠키, 데이터가 모두 내 컴퓨터에 안전하게 보관.

📈

노출 트래킹

작성한 댓글이 노출되고 있는지 자동 확인. 댓글 순위 체크와 중복 영상 스캔까지.

🛠

웹 대시보드

localhost:5000에서 모든 걸 관리. 계정 상태, 작업 현황, SMM 잔액, 설정을 한눈에.

다운로드

CommentBoost ZIP 파일을 다운받고 원하는 폴더에 압축을 풀어주세요.

설치

install.bat을 더블클릭하면 Python과 필요한 모듈이 자동 설치됩니다.

실행

start.bat을 더블클릭하면 브라우저에서 대시보드가 열립니다.

설정

라이선스 키 입력 → Notion 연결 → 유튜브 계정 추가하면 끝!

자주 묻는 질문

클라우드 서비스인가요? ▼
아닙니다. CommentBoost는 내 PC에서 직접 실행되는 프로그램입니다. 유튜브 로그인, 댓글 작성, 브라우저 조작이 모두 내 컴퓨터에서 이뤄지므로 계정 정보가 외부 서버로 전송되지 않습니다. 라이선스 검증과 결제만 클라우드에서 처리됩니다.
Mac에서도 사용할 수 있나요? ▼
현재는 Windows를 공식 지원합니다. Mac/Linux에서도 Python이 설치되어 있다면 수동으로 실행할 수 있지만, ADB IP 변경 등 일부 기능은 Windows 환경에 최적화되어 있습니다.
유튜브 계정이 정지되지 않나요? ▼
CommentBoost는 안전을 최우선으로 설계되었습니다. 안티디텍트 지문, 인간형 랜덤 딜레이, 계정별 IP 분리, 1일 댓글 제한, 같은 영상 간격 제한 등 다중 안전장치가 적용됩니다. 다만, 자동화 도구의 특성상 100% 보장은 할 수 없으며 적절한 설정과 운영이 중요합니다.
Notion이 필수인가요? ▼
네, 현재 버전은 Notion DB를 작업 관리 도구로 사용합니다. Notion 무료 계정으로 충분하며, 제공되는 템플릿을 복제하면 바로 시작할 수 있습니다.
라이선스 키는 어떻게 받나요? ▼
프로그램 설치 후 대시보드에서 원하는 플랜을 선택하고 결제하면 라이선스 키가 자동으로 발급됩니다. Free 플랜은 라이선스 키 없이 바로 체험할 수 있습니다.
