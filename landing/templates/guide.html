설치 가이드

Windows PC에서 CommentBoost를 설치하고 실행하는 방법을 안내합니다.

사전 준비

CommentBoost를 실행하려면 다음이 필요합니다:

  • Windows 10/11 (64비트)
  • 인터넷 연결
  • Notion 계정 (무료 계정 가능)
  • 유튜브 계정 (댓글 작성용)
Python이 설치되어 있지 않아도 괜찮습니다. 설치 스크립트가 자동으로 설치합니다.

Step 1. 다운로드 및 압축 해제

1 CommentBoost ZIP 파일을 다운로드합니다.

2 다운로드한 파일을 원하는 폴더에 압축 해제합니다.

    예: C:\CommentBoost\

폴더 경로에 한글이나 공백이 있으면 오류가 발생할 수 있습니다.
C:\CommentBoost 같은 간단한 경로를 권장합니다.

Step 2. 설치 (install.bat)

1 압축 해제한 폴더에서 install.bat 파일을 더블클릭합니다.

2 검은색 창(명령 프롬프트)이 열리며 자동으로 설치가 진행됩니다.

3 "설치 완료!" 메시지가 나오면 아무 키나 눌러 닫습니다.

설치되는 항목:

  • Python 3.11 (없는 경우 자동 설치)
  • 필요한 Python 패키지 (Flask, Playwright, Notion SDK 등)
  • Chromium 브라우저 (Playwright용)
  • 기본 설정 파일 생성
설치에 약 3~5분 정도 소요됩니다. 인터넷 속도에 따라 다를 수 있습니다.

Step 3. 실행 (start.bat)

1 start.bat을 더블클릭합니다.

2 서버가 시작되면 자동으로 브라우저에서 대시보드가 열립니다.

3 주소: http://localhost:5000

대시보드가 열리면 먼저 회원가입을 진행합니다.

Step 4. 초기 설정 (셋업 위자드)

회원가입 후 자동으로 셋업 위자드가 시작됩니다.

4-1. Notion 연결

  1. Notion 템플릿 복제: 제공되는 링크를 클릭하여 작업 관리용 Notion DB를 복제합니다.
  2. Notion API 토큰 발급:
    • notion.so/my-integrations 접속
    • "새 통합 만들기" 클릭 → 이름 입력 → "내부 통합" 선택
    • 생성된 시크릿 키를 복사
  3. Notion DB에 통합 연결:
    • 복제한 Notion DB 페이지 우측 상단 "..." → "연결" → 생성한 통합 선택
  4. 대시보드에 입력: API 토큰과 Database ID를 붙여넣기합니다.
    • Database ID는 Notion DB URL에서 확인: notion.so/xxxxxxxxxxxxxxxx?v=...

4-2. 유튜브 계정 추가

  1. 대시보드에서 "계정 추가" 클릭
  2. 유튜브(구글) 계정 이메일과 라벨 입력
  3. 처음 실행 시 브라우저가 열려 수동 로그인이 필요합니다
  4. 로그인 후 쿠키가 자동 저장되어 이후에는 자동 로그인됩니다
구글 2단계 인증이 켜져 있으면 첫 로그인 시 인증이 필요합니다.
브라우저에서 직접 인증을 완료해주세요.

4-3. 라이선스 키 입력 (유료 플랜)

무료 체험은 라이선스 키 없이 사용 가능합니다.
유료 플랜으로 업그레이드하려면 대시보드 설정에서 라이선스 키를 입력하세요.

Step 5. 작업 실행

  1. Notion DB에 작업 입력: 영상 링크, 댓글 원고, 계정을 입력하고 상태를 "대기"로 설정
  2. 대시보드에서 "자동화 시작" 클릭
  3. 프로그램이 자동으로:
    • 대기 중인 작업을 Notion에서 불러오고
    • 안전 규칙을 검사한 후
    • 브라우저를 열어 댓글을 작성하고
    • 결과 URL을 Notion에 기록합니다

ADB IP 변경 (선택사항)

안드로이드 폰의 USB 테더링으로 IP를 자동 변경하는 고급 기능입니다.

  1. 안드로이드 폰을 USB로 PC에 연결
  2. 폰에서 "USB 테더링" 활성화
  3. ADB 설치: Android Platform Tools 다운로드
  4. 대시보드 설정에서 "ADB IP 변경" 활성화
  5. 계정이 바뀔 때마다 자동으로 비행기모드 토글 → IP 변경
ADB IP 변경은 필수가 아닙니다. 프록시를 사용하거나, 수동으로 IP를 변경해도 됩니다.

업데이트

update.bat을 더블클릭하면 최신 버전으로 업데이트됩니다.

문제 해결

"Python을 찾을 수 없습니다" 오류

install.bat을 다시 실행해주세요. Python이 자동으로 설치됩니다.

브라우저가 열리지 않음

대시보드 설정에서 "Headless 모드"를 끄고 다시 시도해주세요.

Notion 연결 오류

  • API 토큰이 올바른지 확인
  • Notion DB에 통합이 연결되어 있는지 확인
  • Database ID가 정확한지 확인 (URL에서 복사)

라이선스 검증 실패

  • 인터넷 연결 상태 확인
  • 라이선스 키가 올바른지 확인
  • 방화벽이 api.commentboost.cloud 접속을 차단하고 있지 않은지 확인
